This project only supports 8gen5 (SM8845) / 8elite5 (SM8850)
If you are a beginner, do not use this project yourself; it is recommended to find a flashing service.

This project requires an already unlocked bootloader or the ability to "jailbreak" for temporary root access.

This project must NOT be used for cheating. Hiding BL state is useless — TEE still exposes the real device identity, which can cause device blacklisting. Moreover, the device TEE keys cannot be replaced like TrickyStore.

## Use Cases:
# Banking apps with strong detection:
However, be aware that root does not reduce device security, but unverified modules/modified ROMs absolutely will.
# Netflix:
It can play normally, but don't think about dumping — all video processing is handled inside TEE and cannot be dumped.
# Gaming:
You won't get a 1-day ban from reports, but do NOT use for cheating. Too many reports will still blacklist the device (traces always remain).
